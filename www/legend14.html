Légendes Top14 :

PTS : Points classement
Bonus : Points bonus (offensifs et défensifs cumulés)
J : Nombre de journées jouées
G : Nombre de matchs gagnés
N : Nombre de matchs nuls
P : Nombre de matchs perdus
p : Points marqués (pour)
c : Points encaissés (contre)
+/- : Différence de points
BO : Points bonus offensifs
BD : Points bonus défensifs
Em : Nombre d'éssais marqués
Ee : Nombre d'éssais encaissés
+/- : Différence d'essais
